10

Modul Aplikasi

💰

Manajemen Keuangan

Budget bulanan, tracking transaksi, pengingat tagihan, dan goal tabungan terintegrasi dalam satu modul.

  • Budget allocation dengan pie chart Livewire
  • Quick-entry transaksi via Alpine modal
  • Bill reminder via Laravel Scheduler
  • Savings progress tracker visual
🎯

Produktivitas & OKR

Kanban board interaktif untuk To-Do, breakdown goal besar menjadi langkah kecil yang terukur.

  • Drag-and-drop Kanban (Alpine.js + Sortable.js)
  • Goal → Step breakdown dengan progress %
  • Priority filter & due date sorting
  • Livewire real-time status update
🔄

Habit & Kesehatan

Matrix kebiasaan harian, log berat badan dengan grafik tren, dan meal planner mingguan.

  • Habit matrix click-to-toggle, auto-save
  • Weight trend chart (Chart.js via Alpine)
  • Meal planner → auto grocery generator
  • Streak calculation & badge milestone
⬡

Dashboard Utama

Command center yang merangkum semua modul dalam satu halaman — keuangan, tasks, habits.

  • Financial overview donut chart
  • Today's focus: tasks + meals hari ini
  • Habit streak mini-matrix
  • Bill due-soon alerts panel
🛒

Grocery & Meal

Meal planner mingguan yang secara otomatis mengkompilasi bahan menjadi grocery list.

  • Weekly meal calendar drag-plan
  • Auto-generate grocery dari meal plan
  • Checklist belanja mobile-friendly
  • Kategori bahan (protein, sayur, buah)
🔔

Notifikasi & Alert

Laravel Scheduler + email/Telegram bot untuk reminder tagihan, overdue tasks, dan habit streak.

Planned v1.1

Livewire Component Patterns

app/Livewire/Finance/QuickEntry.php
class QuickEntry extends Component { public string $description = ''; public int $amount = 0; public string $type = 'expense'; protected $rules = [ 'description' => 'required|max:255', 'amount' => 'required|numeric|min:1', ]; public function save(): void { $this->validate(); Transaction::create([ 'user_id' => auth()->id(), 'description' => $this->description, 'amount' => $this->amount, 'type' => $this->type, ]); $this->dispatch('transaction-saved'); $this->reset(); } }
